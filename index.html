Code examples

Task description: Simple, given a string of words, return the length of the shortest word(s). String will never be empty and you do not need to account for different data types.

            function findShort(s) {
              let res = s.split(' ')[0].length

              s.split(' ').forEach((w) => {
                if (w.length < res) res = w.length
              })

              return res
           }
